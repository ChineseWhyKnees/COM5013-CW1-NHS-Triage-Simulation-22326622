# NHS Triage System Simulation

**Student ID:** 22326622
**Status:** In Development

## Overview
A "First Principles" implementation of a clinical triage system.
This project avoids standard Python libraries to demonstrate manual memory management and algorithmic complexity.

## Development Roadmap
 [x] **Stage 1:** Atomic Foundation (Node Primitive & LCG Engine)
 
 [x] **Stage 2:** Linear Data Structures (LinkedQueue, LinkedStack, CircularQueue)
 
 [x] **Stage 3:** Data Model (Immutable PatientRecord with Audit Trail)
 
 [x] **Stage 4:** Logic Layer (Recursive Merge Sort & Chained HashMap)
 
 [ ] **Stage 5:** Abstraction Layer (Patient Factory Pattern)
 
 [ ] **Stage 6:** System Integration (CLI & Recursion Configuration)
 
 [ ] **Stage 7:** Modular Refactoring (Package Architecture)
